4 IP Mulicast Set (IMS) of Rules

  • Context of use: IP multicast is available.
  • Clue: A splitter sends chunks to peers using an IP multicast channel and therefore, peers do not need to communicate each other.
  • Functionality: Provides a buﬀering technique to hidden the (physical) network jitter.
